PWP Milestone 1

Purpose

My PWP will be a portfolio site to supplement my resume and attract potential clients/employers. It will replace the current GitHub Sites version of my portfolio, and will probably incorporate my Java projects as well.

As a stretch goal or post-bootcamp project, it may also link to a functioning interface for my personal project from Deep Dive Java.

Audience

The target audience for my website is recruiters, hiring managers, and clients.

Goals

  • To showcase my programming work and self-advertise.
  • To acquire full-time or freelance employment.
